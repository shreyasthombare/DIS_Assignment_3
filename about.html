About Project

In this project we will develop a web application which will allow user to search for recreation activities in Montgomery County Recreation. The web application will enable user to sign up for a membership for specific recreational activity.

Content

In this assignment we have downloaded the data from data.gov website. The dataset includes details of Montgomery County Recreation activities. The dataset contans columns such as Activity, Recreation Cneter, Time, Age group, Week of the day etc.

Usability

In this assignment we have used chart.js to picturize number of recreational activities through out Montgomery County Recreation. We have summarised our dataset in short which gives an idea of how our dataset look like. We have focused on tables and images for better usability.

Responsiveness

In this assignment we have used viewport and "meta" tag in order to make table and images responsive.

Learning

This assignment helped us to understand basics of web design. We approached this assignment by making wireframes and imlementing them considering usability in mind. This assignment enabled us to gain of javascript, chart.js, html and css.

Contribution of the group members

We worked collaboratively while working on this assignment. We first searched dataset on data.gov website. We worked collaboratively while designing and developing each web page.
